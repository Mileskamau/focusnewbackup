Base URL
- Configurable at runtime from app settings / shared preferences key: api_base_url
- Environment fallback: API_BASE_URL from .env
- Hardcoded fallback if neither is set: http://192.168.100.15/focus8API

Common Headers Sent On Requests
- Content-Type: application/json
- Accept: application/json
- fSessionId: <stored session id>
  This header is added automatically after a successful login if a session id exists in secure storage.

Notes
- All URLs below are relative to the active base URL.
- GET requests usually send no JSON body.
- Some endpoints accept optional query parameters. Those are listed under URL or Body/Query.
- /Transactions/Sales Orders accepts a dynamic body passed in from the caller; the exact payload shape is not defined inside ApiService.
